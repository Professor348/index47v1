__Android Studio__ is the official Integrated Development Environment (IDE) for Android app development, built on [__JetBrains IntelliJ IDEA__](https://www.jetbrains.com/idea/). In FTC, Android Studio is used to write, build, and deploy __Java__ or __Kotlin__ robot control code to the Robot Controller phone or Control Hub. It provides tools like __Gradle__-based build automation, an emulator, debugging, and version control integration. While many teams start with [__Blocks__](https://ftc-docs.firstinspires.org/en/latest/blocks_tutorial/blocks-tutorial.html) or OnBot Java, __Android Studio__ offers full flexibility for advanced teams — including custom __libraries__, multi-module projects, and direct hardware abstraction layer access through the [__FTC SDK__](https://github.com/FIRST-Tech-Challenge/FtcRobotController)
