---
title: Pull Request
aliases: [pr]
---

A __Pull Request__ (PR) is a request to merge code changes from one branch or fork into another branch in a Git repository, usually after review and testing. For example, a tester may find an issue and create code changes to fix it. Other developers can then review the changes before they are officially added to the main code.
