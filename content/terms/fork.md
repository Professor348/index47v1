__Fork__ is a way for programmers to contribute. It is a personal copy of someone else's repository that lives under your own account. It allows you to freely experiment with and modify a project without affecting the original (upstream) version. A fork can be later used to make a pull request for the original repository owner can later choose to accept and merge it or to decline
